# Getting Started

----

## 1. Create a new project

Once you are logged in, please create your project by clicking the "Create new project" button.

<img src="/assets/getting-started/1.png" style="width:100%"/>

## 2. Create a new scope

Once the project created, you'll be asked to create a scope.

<img src="/assets/getting-started/3.png" style="width:100%"/>

## 3. Create a scenario

Then create a scenario, either with "Create scenario" button or small "plus" icon in the scope.

<img src="/assets/getting-started/5.png" style="width:100%"/>

Please write down your test scenario in detailed manner, then click "Create".

<img src="/assets/getting-started/6.png" style="width:100%"/>

At this point, your project is ready to test with a minimum test scenario. Let's test it!

<img src="/assets/getting-started/7.png" style="width:100%"/>

## 4. Conduct new testing session

Please go to the "Session" tab, then click "Create session".

<img src="/assets/getting-started/8.png" style="width:100%"/>

Write down a version that represent this test session then select the scenario that you want to test, or you may want to click "Select all scope" handy shortcut that will select all scenarios in all scopes. Then click "Create"

<img src="/assets/getting-started/9.png" style="width:100%"/>

Your test session item will be appears in Sessions tab. Klik "Join" to start the test.

<img src="/assets/getting-started/10.png" style="width:100%"/>

You can select a scenario and start testing.

<img src="/assets/getting-started/11.png" style="width:100%"/>

Let's say that the test is fail, click "Fail" and write down the reason (if any).

<img src="/assets/getting-started/12.png" style="width:100%"/>

You'll be redirected back to test session page and your scenario will be marked as fail.

<img src="/assets/getting-started/14.png" style="width:100%"/>

That is it!
